🐍 Fundamentos de Python

1. Variables y Asignación

Una variable es un espacio en memoria donde guardamos datos. En Python, se crean simplemente con =:

nombre = "Ana"
edad = 20
precio = 9.99
aprobado = True
💡 Tip: No necesitas declarar el tipo de dato. Python lo detecta automáticamente según el valor que asignes.

3. Entrada de Datos: input()

La función input() lee texto desde el teclado. Siempre devuelve un string, incluso si el usuario escribe un número:

nombre = input("¿Cómo te llamas? ") # devuelve str
edad = input("¿Cuántos años tienes? ") # también devuelve str (¡ojo!)
⚠️ Cuidado: Si quieres hacer operaciones matemáticas con lo que ingresa el usuario, debes convertirlo a número con int() o float().

4. Conversión de Tipos (Parseo)

Para convertir entre tipos de datos usamos funciones especiales:

edad_texto = input("Edad: ") # "25" (es un string)
edad_numero = int(edad_texto) # 25 (ahora es int)

# También se puede hacer en una línea:
edad = int(input("Edad: "))
FunciónConvierte aEjemplo
int()Enteroint("42") → 42
float()Decimalfloat("3.14") → 3.14
str()Textostr(42) → "42"

5. Salida de Datos: print()

print() muestra información en la consola. Puede recibir varios argumentos separados por coma (los separa con un espacio automáticamente):

print("Hola") # Hola
print("Edad:", 25) # Edad: 25
print("Hola", "mundo", "!") # Hola mundo !

🔷 Diagramas de Flujo

Un diagrama de flujo es una representación visual de un algoritmo. Usa figuras estándar para representar cada tipo de operación:

Figuras que usamos

FiguraNombreSe usa para
⬭ ÓvaloInicio / FinMarcar el inicio y el final del algoritmo
▭ RectánguloProcesoOperaciones, cálculos, asignaciones de variables
▱ ParalelogramoEntrada / SalidaLeer datos del usuario (input) o mostrar resultados (print)
◇ RomboDecisiónPreguntas que se responden con Sí o No (compuertas lógicas)

¿Cómo leer un diagrama de flujo?

  • Empieza siempre por el óvalo de Inicio
  • Sigue las flechas en orden descendente
  • En un paralelogramo de entrada, simula que el usuario escribe el valor dado
  • En un rectángulo, realiza la operación indicada
  • En un rombo, evalúa la condición: si es verdadera sigue por "Sí", si es falsa sigue por "No"
  • El paralelogramo de salida muestra el resultado en pantalla
  • Termina en el óvalo de Fin
💡 Consejo práctico: Usa lápiz y papel. Escribe una tabla con las variables y ve actualizando sus valores línea a línea mientras sigues el flujo. ¡Es la técnica más efectiva!

🧠 ¿Cómo resolver estos ejercicios?

Sigue estos pasos para trazar cualquier código o diagrama de flujo:

  • Paso 1: Identifica todas las variables del programa
  • Paso 2: Crea una tabla en tu cuaderno con una columna por variable
  • Paso 3: Ejecuta línea por línea, actualizando los valores de las variables
  • Paso 4: Cuando encuentres un print() o una salida, anota lo que se mostraría
  • Paso 5: Compara tu resultado con las alternativas
Ejemplo de tabla de trazado:
───────────────────────────────
Línea │ a │ b │ resultado
───────────────────────────────
1 │ 5 │ - │ -
2 │ 5 │ 3 │ -
3 │ 5 │ 3 │ 8
4 │ → Imprime: "El resultado es: 8"
