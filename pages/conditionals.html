📖 ¿Qué es un condicional?

Un condicional permite que tu programa tome decisiones. Dependiendo de si una condición es verdadera o falsa, se ejecuta un bloque de código u otro.

💡 Piénsalo así: Es como un semáforo 🚦 — si está en verde, avanzas; si está en rojo, te detienes.

🔹 Estructura: if

La forma más simple. Se ejecuta el bloque solo si la condición es True:

if condición:
# Este código se ejecuta si la condición es True
print("¡La condición se cumplió!")

Ejemplo concreto:

edad = 20

if edad >= 18:
print("Eres mayor de edad")

# Resultado: Eres mayor de edad
⚠️ ¡No olvides los dos puntos : al final de la línea del if! Y el bloque debe estar indentado (con 4 espacios).

🔹 Estructura: if / else

Cuando necesitas un "si no" — un camino alternativo cuando la condición es False:

edad = 15

if edad >= 18:
print("Eres mayor de edad")
else:
print("Eres menor de edad")

# Resultado: Eres menor de edad
¿La condición es True?¿Qué se ejecuta?
Sí (True)El bloque del if
No (False)El bloque del else

🔹 Estructura: if / elif / else

Cuando hay múltiples opciones. elif es la abreviación de "else if" (si no, pero si...):

nota = 5.5

if nota >= 6.0:
print("Excelente")
elif nota >= 5.0:
print("Bien")
elif nota >= 4.0:
print("Suficiente")
else:
print("Reprobado")

# Resultado: Bien
🧠 Regla clave: Python evalúa las condiciones de arriba hacia abajo. Ejecuta el primer bloque True que encuentre y salta todo lo demás. ¡Solo uno se ejecuta!

🔹 Condicionales Anidados

Puedes poner un if dentro de otro if para tomar decisiones más complejas:

edad = 25
tiene_carnet = True

if edad >= 18:
if tiene_carnet:
print("Puede conducir")
else:
print("Necesita sacar el carnet")
else:
print("Es menor de edad")

# Resultado: Puede conducir
💡 Tip: Cada nivel de indentación marca un nuevo nivel de decisión. Usa lápiz y papel para seguirlos.

🔹 En Diagramas de Flujo

Las decisiones se representan con un rombo ◇. Del rombo salen dos caminos: Sí (True) y No (False).

🗝️ Clave para el taller: En los ejercicios de diagramas, sigue siempre el camino que corresponda según el valor de la condición. Si hay múltiples rombos, evalúa cada uno en orden.
