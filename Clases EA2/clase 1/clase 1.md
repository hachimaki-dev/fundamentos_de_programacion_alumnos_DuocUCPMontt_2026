Estructura de datos 

En Python las estructuras de datos se conocen como listas, en otros lenguajes se conoce como array / arraylist

Lo ideal para el código es que las listas estés clasificadas, de acuerdo al tipo de dato que se solicita.

Las listas tienes posiciones, estas posiciones se cuentan desde el 0.
Ej.

[31, 21, 27, 63]
  0   1   2   3

El problemas de las listas es que son inmantenibles, debido a que en la realidad se usan datos de cientos/ miles/ millones de usuarios.
