💻 Calculadora Básica con Manejo de Errores

El manejo de errores en Python permite evitar que un programa falle inesperadamente usando try/except, mejorando la estabilidad y experiencia del usuario.

⚔️ Antes vs Después

❌ Código sin manejo de errores

print("Calculadora básica")

a = int(input("Número 1: "))
b = int(input("Número 2: "))

print("Resultado:", a / b)

Este código es frágil. Si el usuario escribe texto o divide por 0, el programa se rompe completamente.

✅ Código con manejo de errores

try:
    a = int(input("Número 1: "))
    b = int(input("Número 2: "))
    print(a / b)

except ValueError:
    print("Ingrese solo números")

except ZeroDivisionError:
    print("No se puede dividir entre 0")

Ahora el programa es seguro: detecta errores, informa al usuario y continúa funcionando.

🧠 ¿Qué mejora?

  • 🛡️ Evita que el programa se cierre inesperadamente
  • 🎮 Permite interacción continua con el usuario
  • ⚠️ Maneja errores comunes como entradas inválidas
  • 💻 Mejora la experiencia general del sistema
