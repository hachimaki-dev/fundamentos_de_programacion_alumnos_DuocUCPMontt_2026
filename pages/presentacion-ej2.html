Juego de Adivinanza

Ejercicio 2 — Evaluación 3 (60 puntos)

random condicionales ciclos funciones

¿Qué vamos a construir? 🏗️

📥 Entrada

El usuario ingresa un rango numérico (límite inferior y superior)

🎲 Proceso

Se genera un número aleatorio par dentro del rango

🎮 Juego

3 intentos para adivinar, con pistas progresivas

Antes de codear... pensemos 🧠

¿Cómo generamos un número aleatorio en Python?

Si el número aleatorio sale impar, ¿cómo lo convertimos en par?

Flujo General del Programa

📥 Pedir rango
→
🎲 Generar número
→
🔧 Ajustar a par
→
🎮 Juego 3 intentos

Cada bloque es un subproblema que resolveremos por separado

Paso 1 — Pedir el rango al usuario

Python import random limite_inferior = int(input("Ingrese límite inferior: ")) limite_superior = int(input("Ingrese límite superior: "))

💡 Clave Convertimos a int porque input() retorna texto

Paso 2 — Generar el número aleatorio

Python numero_aleatorio = random.randint(limite_inferior, limite_superior) numero_secreto = numero_aleatorio

¿Qué pasa si randint genera un número impar?

Ajuste a número par 🔧

¿numero_secreto % 2 != 0? → Es impar
→
✅ numero + 1 ≤ límite superior
→ sumar 1
❌ se sale del rango
→ restar 1

Paso 3 — Código del ajuste

Python # Si es impar, ajustamos if numero_secreto % 2 != 0: if numero_secreto + 1 <= limite_superior: numero_secreto = numero_secreto + 1 else: numero_secreto = numero_secreto - 1

Ahora viene el juego... 🎮

¿Qué estructura usarías para repetir algo exactamente 3 veces?

¿Cómo saber si el usuario adivinó antes de agotar los intentos?

¿Cómo dar pistas diferentes en cada ronda?

Lógica del juego por ronda

Ronda 1

  • Pedir número
  • ¿Adivinó? → Gana
  • ¿No? → Pista: mayor/menor
  • Guardar este valor

Ronda 2

  • Pedir número
  • ¿Adivinó? → Gana
  • ¿No? → Pista: mayor/menor
  • Pista extra: cercanía

Ronda 3

¿Adivinó? → Gana · ¿No? → Perdiste + revelar número

🔄

Lógica 1: Ciclo for

Enfoque clásico y estructurado

for numero_de_intento in range(1, 4)

Lógica 1 — Ciclo for

Python valor_primer_intento = 0 for numero_de_intento in range(1, 4): valor = int(input(f"Ronda {numero_de_intento}: ")) if valor == numero_secreto: print("¡Felicitaciones!") break elif numero_de_intento == 3: print(f"Perdiste. Era: {numero_secreto}") else: if valor > numero_secreto: print("Es menor") else: print("Es mayor") if numero_de_intento == 1: valor_primer_intento = valor elif numero_de_intento == 2: print(f"Pista: más cerca de {valor}")

Reflexionemos sobre el for 💬

¿Por qué usamos range(1, 4) y no range(0, 3)?

¿Qué hace break dentro del for?

Lógica 2: While + Listas

Enfoque de historial

historial_de_valores = []

¿Cómo funciona el historial?

📋 historial = [] → lista vacía al inicio
→
Ronda 1: usuario ingresa 3 → historial = [3]
→
Ronda 2: usuario ingresa 8 → historial = [3, 8]
Pista: historial[1] vs historial[0]

Lógica 2 — while + Listas

Python intentos_consumidos = 0 historial = [] while intentos_consumidos < 3: ronda = intentos_consumidos + 1 valor = int(input(f"Ronda {ronda}: ")) historial.append(valor) intentos_consumidos += 1 if valor == numero_secreto: print("¡Felicitaciones!") break if intentos_consumidos == 3: print(f"Perdiste. Era: {numero_secreto}") else: if valor < numero_secreto: print("Es mayor") else: print("Es menor") if intentos_consumidos == 2: print(f"Pista: más cerca de {historial[1]}")

Reflexionemos sobre while + listas 💬

¿Cuál es la diferencia entre for y while aquí?

¿Por qué la lista es mejor que una variable para guardar intentos?

¿Qué pasa si olvidamos intentos_consumidos += 1?

Lógica 3: Funciones

Enfoque modular — sin ciclos

def ejecutar_juego(numero_secreto)

¿Por qué funciones?

📦 Encapsulación

Todo el juego vive dentro de una sola función

🚪 return

Usamos return para terminar la función al adivinar

📖 Legibilidad

Cada ronda es un bloque visible y claro

Lógica 3 — Funciones (fragmento)

Python def ejecutar_juego(numero_secreto): # --- RONDA 1 --- valor = int(input("Intento 1: ")) if valor == numero_secreto: print("¡Adivinaste en el 1er intento!") return # pista mayor/menor... primer_intento = valor # --- RONDA 2 --- valor = int(input("Intento 2: ")) if valor == numero_secreto: print("¡Adivinaste en el 2do intento!") return print(f"Pista: más cerca de {valor} que de {primer_intento}") # --- RONDA 3 --- valor = int(input("Intento 3: ")) if valor == numero_secreto: print("¡Adivinaste en el 3er intento!") return print(f"Perdiste. Era: {numero_secreto}") ejecutar_juego(numero_secreto)

Reflexionemos sobre funciones 💬

¿Qué diferencia hay entre break y return?

¿Por qué pasamos el número secreto como parámetro y no lo usamos directo?

¿Qué ventaja tiene esta versión si el juego tuviera 10 intentos?
