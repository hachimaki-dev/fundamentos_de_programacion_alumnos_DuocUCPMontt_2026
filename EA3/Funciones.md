FUNCIONES

Una función es una caja que hace cosas, esas cosas que hace las definimos nosotros.
La funcion tiene datos de entrada, se llaman parametros.
funcion(parametros)

Las funciones tiene una entrada proceso y salida.

Existen todos los lenguajes de programación, pero en Python es el único lugar donde se escribe raro

def saludar(nombre):
    print(f"Hola {nombre}")

saludar("Zobeiba")

Argumento: es el dato concreto ---> Zobeiba-
Parametro: es el nombre del dato que quiero ingresar en la función ---> Nombre

A las funciones le puedo pasar cualquier tipo de dato owo

def saludar(nombre):
    print(f"Hola {nombre}")
    return "chao"

saludar("Zobeiba")

Return es la respuesta de la funcion.

No es necesario que las funciones devuelvan una respuesta, tampoco es obligatorio que tengan un argumento ni parametro.
