📖 ¿Qué es un ciclo while?

Un while repite un bloque de código mientras una condición sea True. Cuando la condición se vuelve False, el ciclo termina.

💡 Piénsalo así: Es como un guardia en una puerta 🚪. Mientras la condición se cumpla, te deja pasar otra vez. Cuando deja de cumplirse, la puerta se cierra.

🔹 Estructura básica del while

El while necesita 3 cosas para funcionar correctamente:

  • Una condición que se evalúa antes de cada vuelta.
  • Un bloque indentado que se ejecuta mientras la condición sea True.
  • Algo que cambie la condición dentro del bloque, para que eventualmente sea False.
🐍Python — while básico
x = 5

while x > 0:     # ← condición
    print(x)     # ← bloque que se repite
    x -= 1       # ← cambio que afecta la condición

# Imprime: 5, 4, 3, 2, 1

🧠 Regla clave: Si nada cambia la condición dentro del while, el ciclo será infinito. ¡Siempre asegúrate de que algo modifique las variables de la condición!

🔹 Ciclo infinito y while True

Si la condición nunca se vuelve False, el ciclo se repite infinitamente. A veces esto es intencional:

🐍❌ Infinito accidental
# ¡CUIDADO! x nunca cambia
x = 5
while x > 0:
    print(x)
# x siempre es 5 → nunca para
🐍✅ Infinito controlado
# while True con break
while True:
    dato = input("Dato (0=salir): ")
    if dato == "0":
        break
    print(f"Procesando: {dato}")
⚠️ Importante: while True: se usa intencionalmente con un break para salir. Sin break, es un error.

🔹 Validar datos con while

Uno de los usos más comunes: pedir datos hasta que sean válidos:

🐍Python — Validación de datos
edad = int(input("Edad: "))

while edad < 0 or edad > 120:
    print("❌ Edad inválida")
    edad = int(input("Edad: "))

print(f"✅ Edad registrada: {edad}")
🧠 Patrón clave: La condición del while describe cuándo el dato es inválido. Mientras sea inválido → pedir de nuevo.

🔹 Patrón centinela

Repetir hasta que el usuario ingrese un valor especial para salir:

🐍Python — Centinela
total = 0
numero = int(input("Número (0 para salir): "))

while numero != 0:
    total += numero
    numero = int(input("Número (0 para salir): "))

print(f"Total: {total}")

🔹 Menú interactivo

Un menú que se repite hasta que el usuario elija salir:

🐍Python — Menú
opcion = ""

while opcion != "3":
    print("1. Saludar")
    print("2. Despedir")
    print("3. Salir")
    opcion = input("Elige: ")

    if opcion == "1":
        print("¡Hola! 👋")
    elif opcion == "2":
        print("¡Adiós! 🌸")

print("Programa terminado")
